About TikkiLabs

TikkiLabs AB is a software studio founded in 2026 and based in Malmö, Sweden. We design, develop, and publish mobile and web applications that solve real, everyday problems across domains like health, productivity, and personal utility.

We believe in building software that is thoughtfully designed, reliable, and genuinely useful. Every product we create starts with a clear purpose and is crafted with attention to quality and simplicity.

What We Build

Mobile Applications

Native iOS applications designed for performance, usability, and seamless integration with platform capabilities.

Web Applications

Responsive, modern web applications built with reliable technologies and a focus on clean user experience.

Health & Wellness

Tools that help people track, understand, and improve their well-being through thoughtful design and useful functionality.

Productivity & Utility

Applications that simplify daily tasks and routines, helping users stay organized and focused on what matters.
